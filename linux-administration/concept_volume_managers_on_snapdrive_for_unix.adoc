= Volume managers on SnapDrive for UNIX
:icons: font
:imagesdir: ../media/

[.lead]

The following table describes the volume managers on your host platform:

a|

a|
Native LVM2

`/dev/mapper/dgname-volname`

`/dev/mapper /dev/dm`
|===
