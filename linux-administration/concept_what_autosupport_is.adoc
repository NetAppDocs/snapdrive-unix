= What AutoSupport is
:icons: font
:imagesdir: ../media/

[.lead]
AutoSupport allows SnapDrive for UNIX to send any operational failures that is incurred due to SnapDrive is sent to the EMS log view of the storage system in the /etc/log/ems file.
