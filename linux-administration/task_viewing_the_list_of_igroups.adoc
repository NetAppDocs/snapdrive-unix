= Viewing the list of igroups
:icons: font
:imagesdir: ../media/

[.lead]
You can view all the igroups that are configured on the host.

. Enter the following command on the host system:
+
`*snapdrive igroup list*`
+
----
snapdrive igroup list
appliance name   igroup name
-----------------------------
toaster        ig
----
